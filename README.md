## üíª Installation

1. Clone this Repository:

```

 $ git clone git@github.com:ruverd/github-explorer.git

```

2. Installing dependencies:

```

$ yarn

```

3. Running project:

```

$ yarn dev

```

You can now view github-explorer in the browser.

Local: http://localhost:3000
